SpringCloud总结

2018年11月11日
| SpringCloud
什么是 SpringCloud Spring Cloud 是一系列框架的有序集合。它利用 Spring Boot 的开发便利性巧妙地简化了分布 式系统基础设施的开发，如服务发现注册、配置中心、消息总线、负载均衡、熔断器、数据 监控等，都可以用 Spring Boot 的开发风格做到一键启动和部署。Spring 并没有重复制造轮子， 它只是将目前各家公司开发的比较成熟、经得起实……
